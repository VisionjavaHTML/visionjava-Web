HTML and CSS basics

Introduction to HTML and CSS

HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are fundamental technologies used to create and design websites. HTML provides the structure and content of a webpage, while CSS controls the presentation and styling.

HTML is composed of elements that define the structure and semantics of a webpage. It uses tags to enclose and define different parts of the content. For example, the <h1> tag is used to define a heading, the <p> tag represents a paragraph, and the <img> tag embeds an image. Elements can also have attributes that provide additional information. For instance, the src attribute in the <img> tag specifies the image source.

CSS, on the other hand, is used to style and format the HTML elements. It allows you to control the appearance of text, layout, colors, and more. CSS works by selecting elements using selectors and applying styles to those elements. Styles can include properties like color, font-size, margin, and many others.

CSS can be applied in different ways. You can use inline styles directly within HTML elements using the style attribute, embed CSS code within a <style> tag in the <head> section of an HTML file, or link an external CSS file using the <link> tag.

By combining HTML and CSS, you can create visually appealing and well-structured webpages. HTML provides the content and structure, while CSS enhances the presentation and layout, allowing you to customize the appearance of your webpages to suit your needs.
